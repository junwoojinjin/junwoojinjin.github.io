## 선정 프로젝트
선정한 프로젝트는 [skku_icampus_auto_downloader](https://github.com/trayo9852/skku_icampus_auto_downloader)이다. 프로젝트이 이름과 같이
성균관대학교 아이캠퍼스(icampus.skku.edu)에서 강의를 다운 받아주는 프로그램이다.

아이캠퍼스 사이트에서 강의를 수강할 때 자유로운 spanning이 어렵고, 시험기간과 같은 중요한 시기에 서버가 터지는 등 강의를 수강하는데 불편함이 많아 기존에도 수동으로 직접 다운 받거나 강의 별로 chrome extension을 사용하여 다운 받는 방법들이 많이 사용되엇다. 
그러나 이러한 방법들은 매 강의마다 사용자가 직접 강의를 재생하고 다운받아야 하며 다운받은 파일을 따로 정리 및 관리 해야하는 것이 매우 번거롭다.
해당 프로젝트는 이러한 불편 사항들을 해결하기 위해 해당 사용자가 수강하는 강좌들을 전부 자동으로 다운받고 강좌별 주차별로 다운받은 파일을 관리해주며 매주 업데이트 되는 강좌들을 자동으로 다운 받아주는 All-in-one 솔루션을 제공한다.


## 프로젝트 분석
선정 프로젝트는 [selenium](https://www.selenium.dev/) (browser automation 라이브러리)와 proxy server와 연결하여 사용 할 수 있는 selenium-wire를 사용하여 구현되어 있다. 
현재는 사용자가 직접 로그인 정보뿐만 아니라, class_id를 지정해주어야하고 아주 기본적인 것들만 구현되어 있는 상태이다.또한 ui적인 측면이나 proxy를 사용하여 속도적인 측면 문제가 있다.


## 프로젝트 개선 사항
아직 사용자 계정에 대한 일반화가 되지 않아 수강하는 강좌의 class id를 직접 설정하여야 하고 UI를 제공하지 않는 등 일반적인 사용자가 사용하기 쉽지 않은 상태이다. 또한 속도를 개선 시킬수 있는 방법이 존재한다.
** 기능 개선 **
1. 다운 받을 위치를 onedrive나 google drive 같은 cloud 서비스를 사용할 수 있도록 기능 추가(저작권 이슈)
2. 자동으로 다운 받은 강의에 대한 볼륨 평탄화/평준화/일반화
3. 강의 다운로드 옵션 추가: 어떤 교과목 강의만 다운 받을지, 몇주차 강의만 다운 받을지, 다운받을 파일명의 format은 어떻게 할지 등 다양한 옵션 선택 가능

** 코드 개선 **
1. 현재는 사용자 계정에 맞게 여러 설정을 하여야 사용이 가능하지만 팀프로젝트를 진행하면서 여러 사용자의 계정에 테스트하여 별도의 설정 없이 로그인 만으로 작동하도록 일반화 시도
2. 기존에는 selenium과 proxy를 사용하여 traffic capture를 사용하여 작동하지만 Session sharing을 통한 direct request를 통해 browser automation에 대한 의존도를 감소시키고 작동 속도 개선
3. 기타 코드 개선 및 모듈화
4. Multiprocessing을 도입하여 다운로드 속도 개선

** UI 개선 및 문서화 **
1. 문서화1: 문서화가 제대로 되어 있지 않아서 해당 프로그램이 제공하는 기능에 대한 자세한 설명과 작동 방법 작성
2. 문서화2: 저작권에 대한 안내 및 사용 시 주의사항에 대한 확고한 공지 추가
2. UI 개선 : CLI 및 GUI를 추가하여 user-friendly 한 UX 제공

## 개인별 프로젝트 기여 계획
** 개발 능력 및 역활 분석 
- 관련 경험 
1. FFmpeg 를 사용한 음원 볼륨 평준화 : 음원의 크기를 맞추기 위해서 FFmpeg 라이브러리를 사용 해본적이 있어, 사용자가 동영상을 다운로드 받고 소리 크기가 다달라서 불편을 겪는 상황을 제거 해 줄수 있을 것이라고 기대한다.

2. Web 개발 경험: Reactjs 를 사용하여 front-end를 제작하고, nodejs로 back-end 서버를 구성하여 AI를 활용한 독서활동이라는 주제로 산학과제를 진행한 적이 있다. 이러한 경험으로 UI 기능을 개선 하는데 도움을 줄 수 있을것이라 생각한다.  또한, Selenium 라이브러리를 사용하는 위 프로젝트의 기능을 개선하는데에, 웹 기반 지식을 활용하여 팀 프로젝트에 기여할 수 있을 것이라 생각한다
